Dark Light

Dark Light lets you decide how each website should look: follow system, preserve site design, force dark, or force light.

How to enable on Mac

  1. Open Safari.
  2. Enable Dark Light in Safari Extensions Preferences.
  3. Use the toolbar popup to switch modes for the current site.

How to enable on iPhone & iPad

  1. Tap the button below to open Settings.
  2. Go to Safari > Extensions.
  3. Tap Dark Light and toggle it On.
  4. Set all website permissions to Allow.

Dark Light is ready to be enabled.
